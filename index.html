Grow with Nigeria Programming Session.

Hello there! This page catalogs all the materials created for the Grow with Nigeria programming session for the 2016 STEM Field Experience program. The session introduces three programming lessons that showcases a wide swath of programming domains from creating web sites, to creating Artificial Intelligence for robots, to creating visualizations for music. Students are paired together and follow the comments in the project files to learn and tackle the programming exercises together

Lesson 1: Intro to Building Websites

Students are paired with a partner to build a website for the fictional company they have started together.

  • Intro to HTML and CSS
  • Create a webpage for your new company
  • Learn what a webpage is, how they are created, the most common HTML tags, and how stylesheets can be used to add swag to HTML pages

Lesson 2: Robocode

RoboCode is a programming game where the goal is to code a robot to compete against other robots in a battle arena. Students will program the Artificial Intelligence (AI) of the robot telling it how to behave and react.

  • Intro to Robocode
  • Create your own champion bot
  • Learn about the Java programming language, what objects, variables, and classes are

Lesson 3: Visualizing music in real time

This lesson using the p5.js Javascript language to display visuals in real-time for a song file. p5.js is a Javascript library that aims to make coding accessible for artists, designers, educators, and beginners.

  • Intro to p5.js
  • What does sound actually look like?
  • Create a sketch that visualizes an input music file
  • Learn about the Javascript programming language, the p5.js library, HTML5 canvas, and creative coding
